# Example app with Storybook + CSS Modules

Using Next.js + Storybook + CSS Modules(.scss).

This repo is based on [with-storybook](https://github.com/vercel/next.js/tree/canary/examples/with-storybook).

## Environment

- Next.js v11.1.2
- Storybook v6.3.11
- CSS Modules(.scss)

## Installation

```
yarn
yarn storybook
```
